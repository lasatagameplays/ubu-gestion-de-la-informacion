### [🤖 Práctica 5 - Filtro colaborativo basado en modelos](./PRACTICAS/Práctica%205%20-%20Filtro%20colaborativo%20basado%20en%20modelos)
* **Directorio:** `/PRACTICAS/Práctica 5 - Filtro colaborativo basado en modelos`
* **Descripción:** Implementación de un sistema de recomendación (filtro colaborativo) utilizando Machine Learning y factorización matricial.
* **Objetivo:** Analizar el dataset MovieLens 100k para entrenar un modelo predictivo basado en el algoritmo SVD. El proyecto abarca la adición de nuevos usuarios, análisis de sesgos, exploración del espacio latente y técnicas de clustering.
* **Tecnologías utilizadas:** Python, Jupyter Notebook, Pandas, Scikit-learn, Surprise, Matplotlib.
* **Estado:** Completada ✔️ *(Calificación: 10/10 - "Muy bien. Solo un comentario, las recomendaciones para nuevos usuarios se hacen sobre una muestra aleatoria, mejor sobre todas las películas para obtener las mejores")*

## 🏆 Proyecto Final

### [🤖 Agentic Workflow (LangGraph)](./Agentic%20Workflow%20(LangGraph))
* **Directorio:** `/Agentic Workflow (LangGraph)`
* **Descripción:** Desarrollo de un agente inteligente que actúa como un sistema de triaje médico automatizado.
* **Objetivo:** Analizar consultas informales de pacientes (ej. síntomas descritos por correo), resumir la información mediante LLMs y utilizar un sistema RAG sobre una base de datos CSV para encontrar la especialidad médica adecuada y proporcionar su teléfono de contacto. Se implementa un grafo de toma de decisiones para validar si la consulta es pertinente o debe ser rechazada.
* **Tecnologías utilizadas:** Python, LangGraph, LangChain, OpenAI API, Jupyter Notebook.
* **Estado:** Completado ✔️ *(Calificación: 8/10 - "Bastante bien, un sencillo workflow con RAG. Áreas de mejora identificadas: limpieza de variables de estado huérfanas tras refactorización y uso más estricto de los umbrales de similitud en el validador")*
